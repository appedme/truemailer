Introduction

Truemailer is a fast, lightweight Email Validation API built with Hono and deployed on Cloudflare Workers. It helps developers and businesses verify emails in real time, detect disposable/temporary addresses, and filter out spammy emails before they enter your system.

Quick Start

To validate an email, simply make a GET request to our validation endpoint:

GET https://api.truemailer.io/validate?email={email}

Example using cURL:

curl "https://api.truemailer.io/validate?email=test@gmail.com"

Validate Endpoint

Request

Method GET
URL /validate
Headers Content-Type: application/json

Query Parameters

Parameter Type Required Description
email String Yes The email address to validate

Responses

200 OK 
{
  "email": "test@gmail.com",
  "valid_syntax": true,
  "domain_exists": true,
  "is_disposable": false,
  "spam_score": 5,
  "verdict": "good"
}
400 Bad Request 
{
  "error": "Email parameter is required"
}

Response Format

Field Type Description
email String The email address that was validated
valid_syntax Boolean Whether the email has valid syntax according to RFC standards
domain_exists Boolean Whether the domain exists and has MX records
is_disposable Boolean Whether the email is from a disposable email service
spam_score Integer Numerical spam score (0-100)
verdict String Human-readable verdict based on spam score

Spam Score Guide

Score Range Verdict Description
0 - 20 Good Valid email with low spam probability
21 - 60 Suspicious Potentially problematic email
61 - 100 Likely Spam High probability of being spam

Code Examples

JavaScript (Fetch API)

async function validateEmail(email) {
  try {
    const response = await fetch(`https://api.truemailer.io/validate?email=${encodeURIComponent(email)}`);
    const data = await response.json();
    
    if (data.verdict === 'good') {
      console.log(`${email} is valid`);
    } else {
      console.log(`${email} might be spam`);
    }
    
    return data;
  } catch (error) {
    console.error('Validation failed:', error);
  }
}

// Usage
validateEmail('user@example.com');

Python (requests)

import requests

def validate_email(email):
    url = "https://api.truemailer.io/validate"
    params = {"email": email}
    
    try:
        response = requests.get(url, params=params)
        data = response.json()
        
        if data["verdict"] == "good":
            print(f"{email} is valid")
        else:
            print(f"{email} might be spam")
            
        return data
    except Exception as e:
        print(f"Validation failed: {e}")

# Usage
validate_email("user@example.com")

Rate Limits

As a public API, Truemailer has rate limits to ensure fair usage:

  • 100 requests per minute per IP address
  • 1000 requests per day per IP address

If you need higher limits for your application, please contact us to discuss enterprise options.
